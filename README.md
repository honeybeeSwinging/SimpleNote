##How to build
1)  Clone the repository

```
$ git clone https://github.com/v2panda/SimpleNote.git
```

2) Cd SimpleNote 

```
pod install
```

3) Open the workspace in Xcode

```
$ open "SimpleNote.xcworkspace"
```

3) Compile and run the app in your simulator or iOS device

##Requirements
* Xcode 7.3+
* iOS8+
